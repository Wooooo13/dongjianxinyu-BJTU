# dongjianxinyu-BJTU 洞见心语 —— 微博热搜话题评论情感分析系统
A project for BJTU

## 项目简介

“洞见心语”是一个基于微博热搜话题的评论情感分析系统，旨在通过对社交媒体用户评论的自动情感识别，帮助用户、决策者、舆情监测机构等快速把握公众情绪与舆论动态。系统结合自然语言处理、机器学习与数据可视化技术，构建了一个直观、可交互的 Web 应用。

## 主要功能

- **实时微博热搜榜展示**：通过微博 API 获取当前热搜话题，支持用户选择感兴趣的话题进行情感分析。
- **评论文本情感分析**：基于 BERT 模型对评论进行情感分类，涵盖七个情感维度。
- **多维度数据可视化**：包括话题热度、情绪分布、地域情绪、点赞互动等图表，采用 ECharts 实现交互式展示。
- **应用案例展示**：提供典型场景下的分析示例，帮助理解系统在实际中的应用价值。

## 技术架构

- **后端框架**：Flask
- **开发语言**：Python
- **情感分析模型**：BERT（Google 开源，fine-tuned 用于中文情感分类）
- **数据获取**：微博 API + Python 爬虫
- **可视化**：ECharts
- **前端**：HTML / CSS / JavaScript

## 系统模块

### 1. 数据获取与清洗
- 使用微博 API 获取热搜榜单
- 编写爬虫获取指定话题下的评论数据
- 数据清洗：去除表情包、颜文字等噪声信息，保留有效文本

### 2. 情感分类
- 预设七类情感维度（如喜悦、愤怒、悲伤等）
- 使用 BERT 模型进行情感倾向判断
- 模型参数调优，提升分类准确率

### 3. 可视化与交互
- 七类可视化图表，包括：
  - 话题类型与热度关系图
  - 情绪分布饼状图
  - 各情绪人数雷达图
  - 热力地图
  - 地区-情绪-点赞数 3D 柱状图
  - 词云图
- 用户可交互切换话题、查看分析结果

## 快速开始

### 环境要求

- Python 3.7+
- Flask
- transformers（BERT 模型）
- 其他依赖详见 `bert_env.yml`

### 运行步骤

1. 克隆仓库：
   ```bash
   git clone https://github.com/yourusername/dongjianxinyu.git
   cd dongjianxinyu
   ```

2. 安装依赖：
   ```bash
   pip install -r bert_env.yml
   ```

3. 启动 Flask 应用：
   ```bash
   python app.py
   ```

4. 访问 `http://localhost:5000` 查看应用
